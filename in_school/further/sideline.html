注意

以下内容仅供参考，具体实际请以当下就业环境为准

通用参考与建议

  1. 推荐大二（因为知识点还没忘光，并且时间也比较充裕）
  2. 推荐个人素质
    1. 耐心（必备）
    2. 体力好
    3. 细心
    4. 良好的心态（因为部分甲方的奇怪要求，比如著名笑话——五彩斑斓的黑）
    5. 交流沟通能力出众（除非干后台）
  3. 推荐的证书
    1. 教资
    2. 普通话
    3. 46级
    4. 计算机二级（含Office / WPS，Python）
    5. 健康证（餐饮兼职必需）
  4. 可能需要携带的证件（需要提供的信息）
    1. 身份证
    2. 学生证
    3. 银行卡号（极少，且绝不会收缴或索取验证码）
  5. 就职前推荐搜索小红书或询问学长该用人单位是否存在压榨学生、不合理收费或拖欠工资等情形

客源

  1. 学长介绍（推荐）
    • 优点：快速入门、不缺单子
    • 缺点：每单将被收取较高的介绍费（约20％～50％）
  2. 在微信、小红书等平台发文自推（不推荐，需要极高信誉、资金、超多时间与广泛的人脉，近乎相当于创业）

形式

  • 体力劳动
    • 优点：不用动脑，大部分情况下也不用和人交流，一般日结；比较放心，几乎没有拖欠工资的可能
    • 缺点：累、时薪极低，通常无法超过25元/时
    • 举例：
      1. 刷盘子等后厨工作
      2. 搬运
      3. 奶茶店、海底捞服务员等服务类行业
      4. 代听讲座、代打卡、代校园跑
      5. 就医陪诊等，详情关注当地医院的招募公告
      6. 跑外卖（没电动车慎选，绝大部分租车合同会规定最低单量，有超高违约金）
  • 脑力劳动
    • 优点：工作取决于你的技能水平与时间。无上限，下限不低于70（低于这个价就是贬低你自己的知识与能力了，没必要接单），时间相对自由
    • 缺点：累（脑力上的），甲方的要求相对较高，且自身能力必须达标，否则会把招牌砸了
    • 举例：
      1. 家教
        • 推荐辅导年级：初二至高二。若认为自己忍耐度较低，请酌情上浮可选年级；否则，可下降年级标准
        • 推荐辅导内容：自己所选的科目，其中中考建议>93分（非赋分制），高考建议>85分（赋分制），推荐理科（文科难见效，家长信任度低）
      2. 中考 /高考咨询
        • 适用人群：当地人、且对升学等有较深刻认识，每年关注升学动态、扩招缩招等公告，且自身分数较高，或与被指导人相近，有指导意义
      3. 科普图绘制 / 科普视频制作
        • 适用人群：熟练掌握剪映（CapCut）/ 达芬奇（DaVinci）/ Adobe After Effects（AE）/ Adobe Premiere Pro（Pr）、Adobe Photoshop（PS）/ Adobe Illustrator（AI）/ Affinity Photo 2 / Affinity Designer 2、Adobe Animate（An）、Blender的使用；有文献查询能力与新领域快速学习能力、能对甲方所需绘制的机制 / 信号通路等示意图有基本认识、有基本审美与逻辑思维、能将需求完善清晰地完成并进行展示
      4. 科研绘图 / 数据统计与分析
        • 适用人群：要求同上。除此以外，要求做到熟悉R语言、Python、SPSS的使用。

绝不参与 / 谨慎参加的项目

  1. 刷单，刷单就是诈骗，没有任何例外
  2. 警惕超级高薪且无技术含量的工作（例如极简单体力劳动，但时薪达90元/时以上的。这种薪资极高且技术含量极低的要么是有毒有害的；要么是违规操作，事后需要找人顶锅的）
  3. 提防需要提前缴费的工作（例如押金、保证金，这种都是你图他回报，他图你本钱）
  4. 绝不参与需要提供银行卡 / 手机号 / 身份证 / 学生证等敏感证件的工作，当心陷入洗钱套路最后因帮信罪连学业都无法完成
  5. 谨慎参与“题库录入”“AI数据标注”以及相关产业，这些产业红利期早已过去，剩下的都是些残羹冷炙。按当下薪资待遇估算，每周工作8小时约获得30元左右，还不如去刷盘子（不要被表面上的高薪迷惑，实际上标注相当麻烦，是要按像素点抠图的，不合格返工后再合格一分钱没有，只有一遍过的给钱，种种套路不一而足，故此建议警惕）……
